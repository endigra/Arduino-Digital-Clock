# Arduino-Digital-Clock
Arduino code and wiring for a Digital clock build using an Arduino Uno R4 Minima
I designed and built a digital clock system using an Arduino Uno R4 Mini, an LCD display, and supporting electronic components. The device displays real-time clock information and includes an alarm feature. When the alarm is triggered, both a buzzer and a vibration motor activate, providing audible and tactile alerts. This project demonstrates my skills in embedded systems, circuit design, and microcontroller programming.
